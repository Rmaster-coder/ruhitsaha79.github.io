# Ruhit Saha Portfolio Website

A clean static portfolio website made for GitHub Pages.

## Files

- `index.html` — main website content
- `styles.css` — design and responsive layout
- `script.js` — mobile menu and scroll animations
- `assets/favicon.svg` — browser icon
- `.nojekyll` — helps GitHub Pages serve the site as plain static files

## How to publish on GitHub Pages

### Option 1: Personal website URL

Use this option if you want your site to be:

`https://ruhitsaha79.github.io`

1. Go to GitHub and create a new repository named exactly:

   `ruhitsaha79.github.io`

2. Upload all files from this folder into the repository.
3. Commit the files.
4. Wait a minute or two.
5. Open:

   `https://ruhitsaha79.github.io`

### Option 2: Project website URL

Use this option if you want to host it inside any normal repository.

1. Create a repository, for example:

   `portfolio`

2. Upload all files from this folder into the repository.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. Your site will normally appear at:

   `https://ruhitsaha79.github.io/portfolio/`
